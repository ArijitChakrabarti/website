# Summary

This repo stores scripts and files supporting my personal website at https://kozodoi.me built using Jekyll and fastpages.

The website includes:
- personal blog with ML tutorials, competition solutions and findings from different projects
- my portfolio with data science projects on different topics
- my academic publications and conference talks with links to the papers and slides
- my Kaggle competition solutions with links to code and write-ups
- summary of my other achievements, including software packages and more
